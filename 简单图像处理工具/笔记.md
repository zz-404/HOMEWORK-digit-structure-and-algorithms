📚 项目中用到的OpenCV库知识详解
1. 核心数据结构 - cv::Mat
cpp
cv::Mat image;  // 图像矩阵对象
作用: OpenCV中最基本的数据结构，用于存储图像数据

特性:

自动内存管理

支持多维数组

多种数据类型支持(CV_8UC1, CV_8UC3等)

2. 图像读取与写入
cpp
// 图像读取
cv::Mat image = cv::imread("path/to/image.jpg");

// 图像写入  
cv::imwrite("output.jpg", image);
imread(): 读取图像文件，支持多种格式(JPEG, PNG, BMP, PPM等)

imwrite(): 保存图像到文件

参数: 可以指定图像质量、压缩格式等

3. 图像显示与窗口管理
cpp
// 创建命名窗口
cv::namedWindow("Window Name", cv::WINDOW_AUTOSIZE);

// 显示图像
cv::imshow("Window Name", image);

// 等待按键
cv::waitKey(0);  // 0表示无限等待

// 关闭窗口
cv::destroyWindow("Window Name");
窗口管理: 创建、显示、销毁图像窗口

waitKey(): 控制窗口显示时间，0=无限等待，正数=毫秒数

4. 颜色空间转换
cpp
cv::cvtColor(inputImage, outputImage, cv::COLOR_BGR2GRAY);
功能: 在不同颜色空间之间转换

常用转换:

COLOR_BGR2GRAY: BGR彩色转灰度

COLOR_GRAY2BGR: 灰度转BGR彩色

COLOR_BGR2RGB: BGR转RGB

COLOR_BGR2HSV: BGR转HSV

5. 图像缩放
cpp
cv::resize(inputImage, outputImage, cv::Size(newWidth, newHeight));
功能: 改变图像尺寸

插值方法:

INTER_NEAREST: 最近邻插值

INTER_LINEAR: 双线性插值(默认)

INTER_CUBIC: 双三次插值

INTER_AREA: 区域插值(缩小图像时效果好)

6. 像素访问与操作
cpp
// 访问单个像素(BGR彩色图像)
cv::Vec3b pixel = image.at<cv::Vec3b>(row, col);
unsigned char blue = pixel[0];
unsigned char green = pixel[1]; 
unsigned char red = pixel[2];

// 修改像素值
image.at<cv::Vec3b>(row, col) = cv::Vec3b(blue, green, red);
Vec3b: 3个unsigned char组成的向量，用于BGR像素

at<>(): 安全访问像素的方法

7. 图像通道与属性
cpp
int rows = image.rows;        // 图像高度(行数)
int cols = image.cols;        // 图像宽度(列数)
int channels = image.channels(); // 通道数
int depth = image.depth();    // 数据类型深度
cv::Size size = image.size(); // 图像尺寸
通道数: 1=灰度, 3=BGR彩色, 4=BGRA(带透明度)

数据类型: CV_8U(8位无符号), CV_32F(32位浮点)等

8. 图像初始化
cpp
// 创建指定大小的黑色图像
cv::Mat image(rows, cols, CV_8UC3, cv::Scalar(0, 0, 0));

// 创建全零矩阵
cv::Mat zeros = cv::Mat::zeros(rows, cols, CV_8UC3);

// 创建全一矩阵  
cv::Mat ones = cv::Mat::ones(rows, cols, CV_8UC3);
9. 标量(Scalar)类型
cpp
cv::Scalar(0, 0, 0)      // 黑色(BGR)
cv::Scalar(255, 255, 255) // 白色(BGR)  
cv::Scalar(0, 0, 255)    // 红色(BGR)
作用: 用于表示颜色值或复数

BGR顺序: OpenCV默认使用BGR颜色顺序，不是RGB

10. 尺寸(Size)类型
cpp
cv::Size size(width, height);
cv::Size size = image.size();
